# JavaWinRing0

> 此项目使用JNA调用WinRing0实现驱动层级键盘模拟，可用于国内网银控件自动输入密码。
> 建议在虚拟机中使用/调试，反正生产环境也是云端虚拟机。
> 欢迎交流：https://www.kagura.me
